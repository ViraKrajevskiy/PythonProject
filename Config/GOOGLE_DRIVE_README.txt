Вложения задач (фото и файлы) сохраняются в Google Drive.

ВАЖНО: У сервисного аккаунта нет своей квоты. Нужна папка внутри
ОБЩЕГО ДИСКА (Shared Drive), а не в «Мой диск».

1. Скопируйте сюда JSON-ключ сервисного аккаунта:
   Config/djangocloud-488316-52ceff61b8f9.json

2. В Google Drive создайте или откройте ОБЩИЙ ДИСК (Shared Drive):
   - Слева: «Общие диски» → «Создать» (или выберите существующий).
   - В общем диске создайте папку для вложений (или используйте корень).
   - ПКМ по общему диску (или по папке) → «Управление участниками».
   - Добавьте: django-drive-bot@djangocloud-488316.iam.gserviceaccount.com
   - Роль: «Менеджер контента» или «Редактор».

3. Узнайте ID папки:
   - Откройте папку в общем диске в браузере.
   - Ссылка вида: .../folders/XXXXXXXXXX
   - Или: .../drive/folders/XXXXXXXXXX
   Скопируйте XXXXXXXX — это GOOGLE_DRIVE_STORAGE_FOLDER_ID.

4. В Config/settings.py (или в переменных окружения) укажите этот ID:
   GOOGLE_DRIVE_STORAGE_FOLDER_ID = 'ID_папки_из_общего_диска'

5. На деплое (если нужно):
   GOOGLE_DRIVE_JSON_KEY_FILE=/путь/к/ключу.json
   GOOGLE_DRIVE_FOLDER_ID=ID_папки_из_общего_диска
